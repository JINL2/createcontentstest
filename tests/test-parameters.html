🎯 다양한 테스트 시나리오

1. 회사만 있는 경우

?user_id=test001&user_name=테스터1&company_id=cameraon

2. 지점만 있는 경우

?user_id=test002&user_name=테스터2&store_id=seoul001

3. 한글 이름 (URL 인코딩)

?user_id=test003&user_name=%EA%B9%80%EC%B2%A0%EC%88%98&company_id=cameraon&store_id=gangnam

4. 베트남어 이름

?user_id=test004&user_name=Nguyen%20Van%20A&company_id=cameraon-vn&store_id=hcm001

💻 개발자용 정보

JavaScript에서 URL 생성하기

const params = new URLSearchParams({ user_id: 'emp001', user_name: '김철수', company_id: 'cameraon', store_id: 'gangnam' }); const url = `${window.location.origin}/mysite/contents_helper_website/?${params.toString()}`;

PHP에서 URL 생성하기

$params = http_build_query([ 'user_id' => 'emp001', 'user_name' => '김철수', 'company_id' => 'cameraon', 'store_id' => 'gangnam' ]); $url = "http://localhost/mysite/contents_helper_website/?{$params}";

브라우저 콘솔 디버깅

// 현재 파라미터 확인 console.log('URL Params:', new URLSearchParams(window.location.search).toString()); // 현재 상태 확인 console.log('Current State:', currentState); // Supabase 연결 테스트 supabaseClient.from('user_progress').select('*').then(console.log);

⚠️ 주의사항

  • user_id는 필수 파라미터입니다. 없으면 자동 생성됩니다.
  • 한글 이름은 URL 인코딩이 필요할 수 있습니다.
  • company_id와 store_id는 데이터 필터링에 사용됩니다.
  • 첫 방문시 user_progress 테이블에 자동으로 사용자가 생성됩니다.
  • localStorage에도 정보가 저장되어 다음 방문시 사용됩니다.
